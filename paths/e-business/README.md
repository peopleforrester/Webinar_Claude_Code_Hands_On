# Path E: Business — Stakeholder Status Report Generator

## Overview

Build a Python tool that reads a structured project status JSON file and generates a professional stakeholder status report in markdown. Executive summary, workstream status table, risks and mitigations, asks from stakeholders, and upcoming milestones — all the things a product manager, engineering manager, or business lead writes every week.

**The important thing:** you don't need to write Python to do this path. You prompt Claude to write the code — you bring business judgment about what a good status report looks like. Claude handles the implementation. The code is incidental; the output is the star.

**Best for:** Product managers, engineering managers, business analysts, operations leads, consultants, anyone who writes recurring status updates for stakeholders.

**Prerequisites:** Python 3.x installed (to run what Claude writes), plus Claude Code CLI. No Python knowledge required from you.
